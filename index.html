Linguagem de programação

HTML

>padrão para a publicação de documentos na web
>delimita e da significado ás informaçoes inseridas na pagina

TODO ARQUIVO HTML DEVE SER UM ARQUIVO DE TEXTO PURO E POSSUIR A EXTENSSÃO '.HTM' OU '.HTML'.

Na pratica:

Tags principais

html e /html>é o elemento “raiz” do documento, delimita o início e o fim da página.

head e /head é o elemento de cabeçalho da página. Por exemplo, é nele que o título será inserido utilizando-se as tags title e /title .

body e /body delimitam o corpo do documento. É no corpo onde iremos teremos o conteúdo em si da página.

Estrutura basica

!DOCTYPE html = evitar que o browser mude para os chamados “quirks mode” quando renderizar um documento. Ou seja, garante que o browser faça um esforço na tentativa de seguir as especificações relevantes, em vez de usar um modo de renderização diferente e que seja incompatível com algumas especificações;

html lang=“pt-br" = especifica o idioma usado em uma página HTML ou em parte dela. Este é usado pelos motores de busca e pelos navegadores;

meta charset="UTF-8" = ao usar o charset dentro da tag meta estamos dizendo ao navegador: “ei, fera, quando for ler esse arquivo (html), use as regras do charset UTF-8”;

meta http-equiv="X-UA-Compatible” content="IE=edge" = para evitar falhas na renderização do site pelo navegador, usar “X-UA- Compatible” com o valor “IE=edge” vai forçar o navegador renderizar com a mais recente do IE (edge);

meta name="viewport" content="width=device-width, initial- scale=1.0"= controla como um site é exibido em um dispositivo móvel.

• width=device-width define o viewport com a largura do dispositivo.
• initial-scale=1.0 define o nível de zoom inicial quando o usuário visita a página.
